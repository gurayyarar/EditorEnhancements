# Editor Enhancements
HTML and URL encodings, transformations and sorting of selected text the editor.

![Demo](https://raw.githubusercontent.com/gurayyarar/EditorEnhancements/master/images/demo.gif)

## Features
- HTML Encode/Decode
- URL Encode/Decode
- To Title Case
- To Lower Case
- To Upper Case
- Capitalize First Letter
- Reverse String
- Remove Accents/Diacritics
- MD5, SHA-1, SHA-256, SHA-384, SHA-512 Hashing
- Line Sorting (Ascending/Descending)
- Remove Empty Lines
- Remove Duplicate Lines
- Remove Horizontal White Space
